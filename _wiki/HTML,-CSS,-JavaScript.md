## clear:both
더 이상 의미 없는 div clear:both를 사용하지 말고 아래 clear fix 이용하라.
```
.clearfix:after {
	content: " ";
	display: block;
	clear: both;
}
```

Less 보다는 Sass를 주로 사용한다고. (전문가를 위한 CSS3, 2013)
